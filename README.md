# Cargocept - Your Trusted Logistics Partner

Professional logistics and courier delivery services worldwide. Fast, reliable, and secure shipping solutions.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd cargocept-connect-flow

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
